BirdsEye is a localised peer reviewed news site that we built in 2 days. It allows for people to post news or community posts,
similar to a grapevine, but all posts have a location, and only people within a set radius can vote on the post, stating if the news is true or not. 
This allows for a peer reviewed system that is far less likely to get votes from people who are not there, as to vote, 
you need to be within the radius of the post. 
 
Coded in Svelte with OpenStreetMap for the frontend, and a PostgreSQL backend database.
